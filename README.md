En este directorio "matias_mantobani_int_an" se encuentra:

1) directorio miaumis: Contiene el programa Python chatbot ayudante para gatos domesticos "miaumis" junto con un archivo setup.txt con las instrucciones de instalacion.

Para ejecutar correctamente el software abrir el directorio miaumis con la consola o IDE de preferencia (recomendamos Visual Studio Code) y seguir las instrucciones del archivo setup.txt contenido dentro de dicho directorio.
